# EcoTrack: Carbon Footprint Awareness Platform 🌍

> **"EcoTrack — A platform to help individuals and campuses measure and reduce their carbon footprint."**

## 📖 Overview

**EcoTrack** is an interactive and user-friendly web application designed to promote environmental sustainability. By providing a streamlined interface to calculate daily carbon emissions, this platform empowers users—ranging from individual households to large university campuses—to track their environmental impact and discover actionable steps to reduce it.

## ✨ Features

* **Individual & Campus Tracking:** Tailored calculation metrics depending on the user's scope (personal vs. institutional).
* **Interactive Footprint Calculator:** Input daily habits (transportation, energy usage, diet) to get real-time carbon footprint estimates.
* **Actionable Recommendations:** Receive personalized tips on how to lower emissions and live more sustainably.
* **Responsive Design:** A clean, accessible interface that works seamlessly across desktop and mobile devices.

## 💻 Technologies Used

This project is built using core frontend web technologies, making it lightweight and easy to deploy:

* **HTML5** (`index.html`): For the structural foundation of the platform.
* **CSS3** (`styles.css`): For styling, layout, and responsive design.
* **Vanilla JavaScript** (`app.js`): For calculator logic, dynamic user interactions, and DOM manipulation.

## 🚀 Getting Started

Follow these step-by-step instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Step 1: Prerequisites
You do not need any complex backend environments or package managers. All you need is:
* A modern web browser (Google Chrome, Mozilla Firefox, Safari, Edge).
* [Git](https://git-scm.com/) installed on your machine (optional, but recommended for cloning).
* A code editor like [VS Code](https://code.visualstudio.com/) (if you plan to edit the code).

### Step 2: Clone the Repository
Open your terminal or command prompt and run the following command to clone this repository to your local machine:
